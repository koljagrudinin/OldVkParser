0. Добавьте базу данных, выполните код из ModelVkontakte.edmx.sql для новой бд, настройте строку подключения для корректного подключения к бд
	1. Строка подключения к БД должна быть одинаковой как в App.Config в проекте ClassLibraryPlanner, так и в проекте WpfApplication1
1. для добавления пользователей и групп скомпилируйте и запустите проект WpfApplication1
	1. Добавьте пользователя, следуя инструкциям на экране
	2. добавьте группы, ВНИМАНИЕ, следует указывать имя группы вконтакте
2. Добавьте службу
	1. Для добавления службы следует :
		1. Скомпилировать проект
		2. Скопировать три файла : 
			1. InstallUtil.exe
			2. InstallUtilLib.dll
			3. InstallUtil.exe.config
			в папку bin проекта WindowsServiceRunner
		3. Открыть командную строку, переместиться в папку bin c помощью команды cd [путь к папке]
		4. выполнить следующую команду
			InstallUtil /i Release\windowsServiceRunner.exe
		5. Панель управления -> Администрирование -> службы, служба называется VKGroupsStatisticAggregator, правой кнопкой -> Пуск
3. Поздравляем! Программа запущена и работает, теперь, если Вы захотите добавить или удалить некую группу, Снова открываете WpfApplication1, и вбиваете туда данные
